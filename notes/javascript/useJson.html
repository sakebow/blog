json数据的使用
作者：柳景兴
时间：2019年2月13日00点25分
关键词：json，特点，使用方法

json格式是JavaScript语法中的一个分支，用法灵活，尤其在网络编程上展现出强大的功能。但是不能识别注释。

json数据的特点

json数据以键值对的形式展现，可以存储极小规模的多种类型变量。 json数据格式最大的便利就是可以给定任意数目、任意类型的数据， 可以是任何一种基本类型，也可以是集合、数组、对象等等。 所以在网络编程的时候经常使用json数据格式传输很多信息。 在使用Python实现网络爬虫的时候，或者使用Java、C#等于语言去实现其他知名公司预留的接口时， 我们的目标往往都是获取json数据，里面会有很多我们想要的东西。

json数据的使用方法

虽然现在已经有很多便利的创建并检查json数据格式的软件，但还是需要说明一些细节问题，便于调试。

首先是：只要是json数据，就必须需要一个大括号包裹起来；这一细节经常会有人忽略，还请多加注意。

其次，键值对的表现形式即为key:value的形式，一般把key叫做“属性”，不同的属性之间使用逗号隔开。 在这里给出一个简单的例子。

自然语言和json数据之间的转化

自然语言：

“大家好，我叫柳景兴，20岁，是一名正在湖北大学计算机与信息工程学院软件工程产业计划专业的大三学生。最近的困扰是买手办被爸妈反对了。”

json数据： 

{
    "人" : {
        "学生" : [{
            "姓名" : "柳景兴",
            "年龄" : 20,
            "学校" : "湖北大学",
            "学院" : "计算机与信息工程学院",
            "专业" : "软件工程产业计划",
            "最近的困扰" : "买手办被爸妈反对"
        }]
    }
}

说明一下，首先是不能掉的大括号。然后确定一个人的分类，将不同的职业确定为这个分类中的不同属性。 每个属性中存储的是一个数组，数组中有很多的个体，我们习惯上叫他“对象”。 对象中的各个属性就是上方的自然语言中提到的各种信息。

而当我们需要获取这些信息的时候，如果是网页上，就需要在script标签中写下： 

人.学生[0].姓名

这样就拿到了"柳景兴"这三个字。
